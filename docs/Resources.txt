Resource Links:
- Open MPW-5 Shuttle Program: https://efabless.com/open_shuttle_program/5 

- SkyWater Open PDK (process design kit – used to create manufacturable designs at SkyWater’s facility): https://github.com/google/skywater-pdk 

- OpenLane RTL2GDS Compiler (an automated RTL to GDSII flow. Performs full ASIC implementation steps): https://github.com/efabless/openlane 

- Caravel Harness (a template SoC for Efabless Open MPW and chipIgnite shuttles based on the Sky130 node from SkyWater Technologies): https://github.com/efabless/caravel 

- Caravel User Project (a sample user project that utilizes the caravel chip user space): https://github.com/efabless/caravel_user_project 

- Open MPW Precheck (checks design/docs using Docker for MPW): https://github.com/efabless/open_mpw_precheck
